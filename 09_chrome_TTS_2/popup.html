About

Voice Assistant v1.0.0
A Chrome extension for Text-to-Speech and Speech-to-Text powered by Google Cloud AI APIs.

  • TTS: Google Cloud Text-to-Speech API
  • STT: Google Cloud Speech-to-Text API
  • Manifest V3 compliant
  • Context menu integration

Enable the APIs in Google Cloud Console and generate an API key with access to both services.
