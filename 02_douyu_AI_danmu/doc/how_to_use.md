# Douyu Danmu Sender 插件使用说明

这是一个用于斗鱼直播间发送弹幕的Chrome插件。

## 1. 安装插件

1.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions` 并回车。
2.  在打开的扩展程序页面，**打开右上角的“开发者模式”开关**。
3.  点击左上角的“**加载已解压的扩展程序**”按钮。
4.  在文件选择对话框中，导航到你的项目目录：`/home/user/userFile/AI-it-is-depends-on-you/02_douyu_AI_danmu/extensions`。
5.  选择 `extensions` 文件夹，然后点击“选择文件夹”。
6.  插件图标会出现在你的浏览器工具栏上。

## 2. 使用方法

1.  **访问斗鱼直播间**：在 Chrome 浏览器中，打开任意一个斗鱼直播间页面（例如：`https://www.douyu.com/` 后面的具体房间号）。
2.  **点击插件图标**：点击浏览器工具栏上的“Douyu Danmu Sender”插件图标。
3.  **输入弹幕内容**：在弹出的窗口中，将你想要发送的弹幕内容输入到文本框中。
4.  **发送弹幕**：点击“发送”按钮。

插件会自动将你输入的内容填入直播间的弹幕输入框，并模拟点击发送按钮，从而发送弹幕。

## 3. 配置管理 (高级功能)

插件支持将配置（如 API Key、AI 提示词等）保存到本地 JSON 文件，方便备份与共享。

1.  **打开设置**：点击插件弹窗右上角的齿轮图标进入设置页面。
2.  **选择文件**：
    *   点击“选择文件”按钮，选择一个已存在的 JSON 配置文件。此时不会修改文件内容。
3.  **新建/保存配置**：
    *   若想新建配置文件，在未选择文件的情况下直接点击“保存到文件”，输入新文件名即可。
    *   若想更新当前选中文件的内容，点击“保存到文件”即可覆盖写入。
4.  **读取配置**：点击“从文件读取”，将文件中的配置加载到插件中。
5.  **自动同步**：勾选“启用自动同步”，插件会自动读取文件配置，且任何修改都会实时保存回文件。

## 4. 注意事项

*   请确保你正在访问斗鱼直播间页面，否则插件可能无法找到目标元素。
*   如果斗鱼网站的HTML结构发生变化，插件可能需要更新选择器才能正常工作。
